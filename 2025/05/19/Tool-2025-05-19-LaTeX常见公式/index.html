LaTeX常见公式

发表于2025-05-19|更新于2025-07-16|数学LaTex
|总字数:759|阅读时长:3分钟|浏览量:

在 LaTeX 中，数学符号的表示依赖于特定的命令和环境（需在数学模式中，如 $...$ 或 \[...\]）。以下是常见数学符号的分类及对应的 LaTeX 代码：

1. 希腊字母

符号 LaTeX 代码 符号 LaTeX 代码
α \alpha Α \Alpha
β \beta Β \Beta
γ \gamma Γ \Gamma
δ \delta Δ \Delta
ε \epsilon Ε \Epsilon
θ \theta Θ \Theta
π \pi Π \Pi
σ \sigma Σ \Sigma
ω \omega Ω \Omega
ζ \zeta Ζ \Zeta

2. 运算符

符号 LaTeX 代码 说明
+ + 加号
− - 减号
× \times 乘法
÷ \div 除法
± \pm 正负号
∓ \mp 负正号
√ \sqrt{x} 平方根
∑ \sum_{i=1}^n 求和
∫ \int_{a}^{b} 积分
∏ \prod_{i=1}^n 连乘积
∂ \partial 偏导数符号
≈ \approx 约等于
≠ \neq 不等于

3. 集合与逻辑符号

符号 LaTeX 代码 说明
∈ \in 属于
∉ \notin 不属于
⊂ \subset 子集
⊆ \subseteq 包含于
∪ \cup 并集
∩ \cap 交集
∅ \emptyset 空集
∀ \forall 全称量词
∃ \exists 存在量词
¬ \neg 非
∧ \land 逻辑与
∨ \lor 逻辑或

4. 箭头符号

符号 LaTeX 代码 说明
→ \to 或 \rightarrow 右箭头
← \leftarrow 左箭头
↔ \leftrightarrow 双向箭头
⇒ \Rightarrow 右双线箭头
⇔ \Leftrightarrow 双向双线箭头
↦ \mapsto 映射箭头

5. 括号与定界符

符号 LaTeX 代码 说明
( ) ( 和 ) 圆括号
[ ] [ 和 ] 方括号
{ } \{ 和 \} 花括号（需转义）
⟨ ⟩ \langle 和 \rangle 尖括号
⎡ ⎤ \lceil 和 \rceil 上取整括号
⎣ ⎦ \lfloor 和 \rfloor 下取整括号

6. 上下标与标注

符号 LaTeX 代码 说明
上标 x^2 x²
下标 x_1 x₁
多字符上标 x^{n+1} xⁿ⁺¹
导数符号 f'(x) 或 f^{\prime} f'(x)
向量箭头 \vec{v} v⃗
顶部横线 \overline{AB} 线段 AB 的标注

7. 矩阵与方程组

矩阵

  • pmatrix: 圆括号矩阵
  • bmatrix: 方括号矩阵

8. 符号字体样式

样式 LaTeX 代码 示例
粗体 \mathbf{A} A
花体 \mathcal{A} 𝒜
黑板粗体 \mathbb{R} ℝ（实数集）
手写体 \mathscr{A} 𝒜（需 mathrsfs 包）

注意事项

  1. 数学模式：所有符号需包裹在 $...$（行内公式）或 \[...\]（独立公式）中。
  2. 包依赖：部分符号需加载 amsmath、amssymb 或 mathrsfs 包。
  3. 符号冲突：避免与 LaTeX 保留字符冲突（如 \、%、& 需转义）。

掌握这些符号的 LaTeX 表示后，可以高效编写数学公式！如需更多符号，可查阅 LaTeX 数学符号大全。
