🧠 Vim 快捷指令速查表

Vim 是一个功能强大的文本编辑器，适合程序员日常高效编辑代码和文本。本速查表适合初学者，也可用于日常参考。

📌 模式概览

模式名称 进入方式 作用
普通模式 启动后默认，或按 Esc 移动、编辑、复制、粘贴等
插入模式 i, a, o 等 输入文字
命令模式 : 保存、退出、查找替换等

✍️ 插入与编辑操作（普通模式）

指令 说明
i 在光标前插入
a 在光标后插入
o 在下方新开一行并插入
O 在上方新开一行并插入
I 跳到行首插入
A 跳到行尾插入
Esc 退出插入模式回到普通模式

🧭 光标移动（普通模式）

指令 说明
h 左移一个字符
l 右移一个字符
j 下移一行
k 上移一行
0 移动到行首
^ 移动到行首第一个非空字符
$ 移动到行尾
w 向前跳一个单词
b 向后跳一个单词
gg 移动到文件第一行
G 移动到文件最后一行
nG 或 :n 跳转到第 n 行

🗂️ 复制、粘贴、删除

指令 说明
yy 复制当前行（yank）
nyy 复制接下来的 n 行
p 在光标后粘贴
P 在光标前粘贴
dd 删除（剪切）当前行
ndd 删除接下来的 n 行
x 删除当前字符
d$ 删除从光标到行尾

🔁 撤销与重做

指令 说明
u 撤销上一个操作
U 撤销当前行的所有修改
Ctrl + r 重做撤销的操作

🔍 查找与替换

命令 说明
/word 向下查找 “word”
?word 向上查找 “word”
n 查找下一个匹配项
N 查找上一个匹配项
:s/旧/新/ 当前行替换第一个匹配项
:s/旧/新/g 当前行替换所有匹配项
:%s/旧/新/g 全文替换所有匹配项
:%s/旧/新/gc 全文替换，逐一确认

💾 文件操作

命令 说明
:w 保存
:q 退出
:wq 或 :x 保存并退出
:q! 强制退出（不保存）
:e 文件名 打开另一个文件

🪟 分屏与标签页

指令 说明
:sp 文件 水平分屏打开文件
:vsp 文件 垂直分屏打开文件
Ctrl + w + h/j/k/l 在分屏窗口间切换
:tabnew 文件 新建标签页并打开文件
gt 下一个标签页
gT 上一个标签页

⚡ 快捷组合示例

操作 快捷指令
保存退出 :wq 或 ZZ
只退出（不保存） :q!
删除当前行并插入新行 cc
替换当前单词 cw
删除当前单词 dw
