ElGamal简介

ElGamal密码体制由T.ElGamal于1985年提出，基于的困难问题是群中的离散对数问题。

  • 离散对数问题
    乘法群Zp∗Z_p^*Zp∗​上的离散对数问题（Discrete Logarithm Problem, DLP）：给定一个素数p，乘法群Zp∗Z_p^*Zp∗​上的生成元ggg，以及Zp∗Z_p^*Zp∗​上的随机选取的元素yyy，寻找整数x,1≤x≤p−1x, 1 \leq x \leq p-1x,1≤x≤p−1，使得y=gxmod  py = g^x \mod py=gxmodp
    易知，Zp∗Z_p^*Zp∗​中元素的个数为p−1p-1p−1，即{1,2,...,p−1}\{1,2,...,p-1\}{1,2,...,p−1}
  • 推广的离散对数问题
    给定循环群GGG，设其阶为nnn，生成元为ggg，及GGG上随机选取的元素y，寻找整数xxx，1≤x≤n−11 \leq x \leq n-11≤x≤n−1，使y=gxy = g^xy=gx
    这个整数xxx记为log⁡gy\log_gylogg​y，称为离散对数

Diffie-Hellman

DHP问题

  • 给定循环群GGG，设其阶为nnn，生成元为ggg，以及GGG上随机选取的元素α=ga,β=g\alpha = g^a, \beta = gα=ga,β=g计算γ=gab\gamma = g^{ab}γ=gab。

Diffie-Hellman密钥交换协议

  • Alice与Bob选择一个大素数ppp，选ggg作为乘法群Zp∗Z_p^*Zp∗​中一个生成元。
  • Alice选一个整数aaa：1≤a≤p−21 \leq a \leq p-21≤a≤p−2，并计算A=gamod  pA = g^a \mod pA=gamodp。发送AAA给Bob。
  • Bob选一个整数bbb：1≤b≤p−21 \leq b \leq p-21≤b≤p−2，并计算B=gbmod  pB = g^b \mod pB=gbmodp。发送BBB给Alice。
  • Alice计算k=Bamod  pk = B^a \mod pk=Bamodp。
  • Bob计算k=Abmod  pk = A^b \mod pk=Abmodp。
