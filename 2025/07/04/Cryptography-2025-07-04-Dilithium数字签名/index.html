格密码学基础

  • 模块格(Module Lattices)定义与性质
    • 传统欧式格：在线性代数中，欧式格(lattice)是指由一组基向量的整数线性组合所生成的离散点集。例如标准格可表示为L={∑i=1naibi:ai∈Z}L = \{\sum_{i=1}^na_ib_i:a_i \in \mathbb{Z}\}L={∑i=1n​ai​bi​:ai​∈Z}。这样的格是自由Z\mathbb{Z}Z-模，其维度为基向量个数。
    • 模块格定义：模块格将欧式格的概念推广到带有环结构的模空间上。具体地，取模数域为多项式环

      Rq=Zq[X]/(Xn+1)R_q = \mathbb{Z}_q[X]/(X^n+1) Rq​=Zq​[X]/(Xn+1)

      其中qqq为素数或整除数(Dilithium中一般取q=223−213+1q = 2^{23} - 2^{13} + 1q=223−213+1)，n=256n = 256n=256（退化多项式Xn+1X^n+1Xn+1）。令RqkR_q^kRqk​表示由k个长度为n的多项式构成的向量空间。由于每个多项式可视作长度nnn的向量，RqkR_q^kRqk​可嵌入为Zqkn\mathbb{Z}_q^{kn}Zqkn​，从而是一个维度为nknknk的自由Z−\mathbb{Z}-Z−模。当以Rq−R_q-Rq​−模方式给出时，这样的格成为模块格。换言之，将传统LWE和SIS问题中的Zqn\mathbb{Z}_q^nZqn​替换为RqR_qRq​上的RqkR_q^kRqk​模，即得到模块学习有误差(MLWE)和模块短整数解(MSIS)问题。
  • 学习带错误问题(MLWE)及其变体
    • 定义：模块学习有误差(Module Learning With Errors, MLWE)是LWE的推广。在环RqR_qRq​上，对于给定的整数参数k,ℓk,\ellk,ℓ和噪声分布D⊂RqD \subset R_qD⊂Rq​，选取随机矩阵A∈Rqk×ℓA \in R_q^{k \times \ell}A∈Rqk×ℓ​，秘密向量s1∈Rqℓ,s2∈Rqks_1 \in R_q^\ell, s_2 \in R_q^ks1​∈Rqℓ​,s2​∈Rqk​从噪声分布D中独立采样，然后构造“有噪声的线性组合”

      t=As1+s2    ( mod   q) .\mathbf{t} = A\mathbf{s}_1 + \mathbf{s}_2 \;\;(\bmod\;q)\,. t=As1​+s2​(modq).

      MLWE问题是：给定样本(A,t)(A,\mathbf{t})(A,t)，判断t\mathbf{t}t是否由上述式子产生，或等价地求解秘密(s1,s2)(\mathbf{s}_1, \mathbf{s}_2)(s1​,s2​)。在Dilithium中，私钥s1,s2\mathbf{s}_1, \mathbf{s}_2s1​,s2​的系数均为小整数（绝对值上届为η\etaη），而公钥t\mathbf{t}t的生成即为

      t=As1+s2 ,s1∈Rqℓ,  s2∈Rqk,  ∣(s1)i,j∣,∣(s2)i∣≤η\mathbf{t} = A\mathbf{s}_1 + \mathbf{s}_2 \,,\quad \mathbf{s}_1\in R_q^\ell,\; \mathbf{s}_2\in R_q^k,\;|(\mathbf{s}_1)_{i,j}|,|(\mathbf{s}_2)_i|\le\eta t=As1​+s2​,s1​∈Rqℓ​,s2​∈Rqk​,∣(s1​)i,j​∣,∣(s2​)i​∣≤η

      以上定义可用判别表述：MLWE问题要求区分“(A,tA,tA,t)为随机无关向量”与“由密钥生成的(A,As1+s2A, A\mathbf{s}_1+\mathbf{s}_2A,As1​+s2​)”两类分布。
    • 参数与噪声分布：MLWE的安全性取决于环RqR_qRq​的维度nnn、矩阵尺寸(k,ℓ)(k, \ell)(k,ℓ)、模数qqq以及噪声范围η\etaη等参数。不同安全级别下k,ℓk, \ellk,ℓ不同。私钥向量系数从均匀分布或截断离散高斯分布D中采样，使得s1,s2\mathbf{s}_1, \mathbf{s}_2s1​,s2​具有小范数。这些参数确保MLWE样本t\mathbf{t}t中掩埋的秘密向量极难被恢复。
    • 困难性来源：MLWE可证明在最坏情况下等价于环上某些标准格问题（如GapSVP）的近似求解，因而难以破解，直观地说，MLWE对应的格在其点集中含有一个“异常短”的秘密向量。MLWE问题相当于在一个格中寻找一个很短的向量，而该格是由随机矩阵A构造的，并且存在一个显著更短的秘密向量。这种“低密度背包”结构使得针对MLWE得攻击难度极高。目前已知的最佳攻击方法是将MLWE视作高维LWE（维度约为nℓn\ellnℓ）并应用BKZ格简约或格筛选算法，其时间复杂度随为度指数增长。在量子模型下，虽可利用Grover或量子筛选获得常数次加速，但总体复杂度依然呈指数级增长，因此MLWE被认为具有抗量子攻击能力。
      1. 格结构：
        • 由矩阵A\mathbf{A}A生成规格晶格
        • 格点间距由qqq和nnn决定
      2. 秘密短向量：e\mathbf{e}e
        • 长度很小
        • 方向随机
        • 连接真实格点As\mathbf{As}As和观测点b\mathbf{b}b
      3. 攻击者视角：
        • 只知道A\mathbf{A}A格结构和b\mathbf{b}b（偏移点）
        • 需要找到As\mathbf{As}As或e\mathbf{e}e
        • 但b\mathbf{b}b被包裹在“错误球”中
    • Dilithium中的应用：在 Dilithium 中，公钥的生成直接采用 MLWE 结构：首先随机生成 A∈Rqk×ℓA\in R_q^{k\times\ell}A∈Rqk×ℓ​，然后从小范围采样秘密 s1∈Rqℓ,s2∈Rqk\mathbf{s}_1\in R_q^\ell,\mathbf{s}_2\in R_q^ks1​∈Rqℓ​,s2​∈Rqk​，并计算 t=As1+s2\mathbf{t}=A\mathbf{s}_1+\mathbf{s}_2t=As1​+s2​。根据 MLWE 假设，(A,t)(A,\mathbf{t})(A,t) 在计算上不可与随机元素区分，这意味着公钥暴露的信息几乎不包含 s1,s2\mathbf{s}_1,\mathbf{s}_2s1​,s2​ 的线索，从而保证了私钥的安全。
  • 短整数解问题(MSIS)安全性基础
    • 定义：模块短整数解 (Module Short Integer Solution, MSIS) 是 SIS 问题在环上的推广。形式化地，给定随机矩阵 A∈Rqm×kA\in R_q^{m\times k}A∈Rqm×k​，考虑增广矩阵 [I∣A]∈Rqm×(m+k)[I\mid A]\in R_q^{m\times(m+k)}[I∣A]∈Rqm×(m+k)​，其中 III 是 m×mm\times mm×m 单位矩阵。MSIS 问题要求在整数环上找到非零向量 y∈Rqm+k\mathbf{y}\in R_q^{m+k}y∈Rqm+k​（向量可分解为 y=(u,v)\mathbf{y}=(\mathbf{u},\mathbf{v})y=(u,v)），满足

      [I∣A]  y=u+Av  ≡  0  ( mod   q) ,且  ∥y∥∞≤γ ,[I\mid A]\;\mathbf{y} = \mathbf{u} + A\mathbf{v} \;\equiv\; \mathbf{0}\;(\bmod\;q)\,,\quad \text{且}\;\|\mathbf{y}\|_\infty \le \gamma\,, [I∣A]y=u+Av≡0(modq),且∥y∥∞​≤γ,

      即求解方程组 u+Av=0\mathbf{u} + A\mathbf{v} = \mathbf{0}u+Av=0（模 qqq）的非平凡短解，其中 ∥⋅∥∞\|\cdot\|_\infty∥⋅∥∞​ 表示无穷范数上界，γ\gammaγ 是给定的长度上限。直观地说，MSIS 需要找到增广线性方程组的一组“系数”(u,v)(\mathbf{u},\mathbf{v})(u,v)，使得 u=−Av\mathbf{u} = -A\mathbf{v}u=−Av 且所有分量都很小。
    • 实例说明：若将 y=(s2, −s1)\mathbf{y}=(\mathbf{s}_2,\,-\mathbf{s}_1)y=(s2​,−s1​) 代入，则有

      [I∣A](s2−s1)=s2−As1≡0(modq)[I\mid A] \begin{pmatrix}\mathbf{s}_2\\ -\mathbf{s}_1\end{pmatrix} = \mathbf{s}_2 - A\mathbf{s}_1 \equiv \mathbf{0} \pmod{q} [I∣A](s2​−s1​​)=s2​−As1​≡0(modq)

      当且仅当 As1+s2≡0A\mathbf{s}_1+\mathbf{s}_2\equiv 0As1​+s2​≡0。然而在签名方案中，s1,s2\mathbf{s}_1,\mathbf{s}_2s1​,s2​ 本身通常较大或具有一定分布，因此找到使 ∥y∥∞≤γ\|\mathbf{y}\|_\infty\le\gamma∥y∥∞​≤γ 的非平凡解等价于破解 SIS。MSIS 要求解的向量必须比简单构造的 (s2,−s1)(\mathbf{s}_2,-\mathbf{s}_1)(s2​,−s1​) 更短，因而问题非平凡。
    • 计算复杂度： MSIS 相当于在高维格中寻找一个非常短的向量，这通常是 NP 难的近似格问题。具体来说，MSIS 对应于在维度 n(k+m)n(k+m)n(k+m) 的格上解 ∥y∥∞≤γ\|\mathbf{y}\|_\infty\le\gamma∥y∥∞​≤γ 的最短向量问题。Ducas 等人指出，MSIS 相当于在随机格中找短向量，是高密度背包问题。当前最佳算法仍是基于格简约（如 BKZ）或格筛选的方法，这些算法时间复杂度随维度呈指数增长，在经典和量子模型下均难以大幅优化。因此，从理论上讲，若参数选择适当，MSIS 也具备很强的抗量子攻击能力。
    • Dilithium中的应用：MSIS 在 Dilithium 中主要用于保证签名的不可伪造性。根据安全性证明，Dilithium 的强不可伪造性可以归约到 MLWE 和 MSIS 两个假设。在签名伪造的情境下，一位攻击者需要找到新的签名 (z,c)(z,c)(z,c) 使得验证条件成立，对应着求解方程

      Az−c t=（低位小扰动） Az - c\,\mathbf{t} = \text{（低位小扰动）}\, Az−ct=（低位小扰动）

      其中 t=As1+s2\mathbf{t}=A\mathbf{s}_1+\mathbf{s}_2t=As1​+s2​ 是公钥。将此方程重写可看作 [I∣A]y=u[I\mid A]\mathbf{y}=\mathbf{u}[I∣A]y=u 的形式，其中 y=(z, −c)\mathbf{y}=(z,\,-c)y=(z,−c)，u\mathbf{u}u 与哈希挑战有关。这恰好是一个带有目标哈希约束的 MSIS 问题（SelfTargetMSIS），也可理解为在 [I∣A][I\mid A][I∣A] 生成的格中求解短向量问题。如果攻击者能成功伪造签名，则等价于破解相应的 MSIS 问题。因此，MSIS 的困难性保证了 Dilithium 签名的强安全性。

总结与展望

遇到的问题

在实现代码的过程中会发现设计Dilithium类的算法非常困难，总是遇到验证失败的过程，后来发现是模运算不一致，中心化处理时机不当和多项式乘法最终没有模q而导致的边界问题。矩阵向量乘法也需要及时取模防止数据溢出。最终也进行了问题的修复。

对未来的展望

在未来，基于MLWE问题，设计的dilithium环签名可以有效抵抗格基约化的攻击，当前有效的攻击也完全可以抵抗，量子算法对MLWE的加速也是有限的，可以很好的抗后量子攻击。
